Featured Project

HLS for E-Bike Control

HLS (High level system) for E-bike, controls all functions of a e-bike using BLE, 2G/3G, and Wifi. Had also GPS, INS with dedicated STMF4 for Kalman and Euler calculations, 2 CANs and Li-ion batteries.

  • Mobile app interface for live diagnostics and control
  • Integrated telematics with GPS and inertial sensing
  • Dedicated MCU path for motion estimation and filtering
  • Multi-bus architecture for controller and battery subsystems

Power Electronics Project

PWR Invertor for E-Bike

Power invertor for e-bike, 100V, up to 300A peak, up to 100A constant, CAN interface, isolated drivers for all main mosfets.

  • 100V drivetrain power stage designed for high-current e-bike operation
  • Up to 300A peak and 100A continuous current capability
  • CAN interface for system-level coordination with vehicle electronics
  • Isolated gate drivers for the main MOSFET switching path
