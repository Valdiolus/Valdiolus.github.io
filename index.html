Project

Rear View Camera Dataset

A perception dataset for smart bicycle rear-view cameras, created to train object-detection models across realistic traffic scenes with both daytime and nighttime conditions.

  • COCODetectionDataset format prepared for FiftyOne-based inspection workflows
  • 32,448 training images and 3,948 validation images
  • Six classes: person, car, bicycle, motorcycle, bus, and truck
  • Used for YOLOv5 training and auto-labeling experiments for rear-view mobility vision systems
Open repository

Project

License Plate Detection Dataset

A dedicated license plate detection dataset built for training deep neural detectors on plate localization tasks in road-scene imagery.

  • COCODetectionDataset structure with FiftyOne-compatible visualization tooling
  • 3,395 training images and 500 validation images
  • Two classes focused on plate detection workflows: plate and background
  • Used to train YOLOv5-based license plate detectors for compact detection pipelines

RWC

RWC is a smart bicycle rear-view camera project developed to improve cyclist safety, combining embedded vision hardware, on-bike recording, custom datasets, and real-time AI deployment.

  • Developed 5 prototypes using Raspberry Pi, Coral AI accelerator, custom photo lenses, microphones, speakers, buzzers, and powerful day/night back lights
  • Built software for video recording on-the-go while cycling
  • Launched beta tests with customers, and used recorded videos to create custom computer-vision datasets
  • Formed an annotation team for labeling the dataset and scaling data preparation
  • Trained neural networks for a real-time embedded AI device using the custom dataset
  • Open-sourced the full datasets with about 40k photos for object detection and about 10k for license plate detection

Project

PWR_invertor

Power invertor for e-bike, 100V, up to 300A peak, up to 100A constant, CAN interface, isolated drivers for all main mosfets.

  • 100V drivetrain power stage designed for high-current e-bike operation
  • Up to 300A peak and 100A continuous current capability
  • CAN interface for system-level coordination with vehicle electronics
  • Isolated gate drivers for the main MOSFET switching path
